## Changes Made

- Added LVGL dependency through ESP-IDF Component Manager:
  - `lvgl/lvgl` resolved to `9.4.0`
- Replaced the full-screen RGB565 bitmap UI with LVGL runtime drawing:
  - black screen background
  - title label
  - connected/disconnected status labels
  - three large C/V/X button panels using LVGL Montserrat fonts
- Removed the generated bitmap source arrays:
  - `main/ui_assets.c`
  - `main/ui_assets.h`
- Kept the original PNG files in `UIresources/` as design references.
- Changed LCD color order to `LCD_RGB_ELEMENT_ORDER_RGB`.
- Adjusted LCD mirroring from `(true, true)` to `(true, false)` while keeping `swap_xy=true`, matching the requested extra rotation/mirror correction as the first test setting.
- Preserved GPIO23 beeper feedback and the existing touch shortcut behavior.
- Reduced LVGL memory usage for ESP32 DRAM:
  - LVGL memory pool: `32KB`
  - simple draw layer buffer: `8192`
  - single partial draw buffer: 12 display lines
- Increased `lvgl_task` stack to `8192` after serial logs showed `lvgl_task` stack overflow at `4096`.
